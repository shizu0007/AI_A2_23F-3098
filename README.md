# Dynamic Pathfinding Agent

AI-2002 Assignment 02 -- Question 6

## Description

This project implements a dynamic grid-based pathfinding system using:

-   A\* Search
-   Greedy Best First Search
-   Manhattan Heuristic
-   Euclidean Heuristic
-   Dynamic Obstacles
-   Real-time Visualization
-   Performance Metrics

## Requirements

Python 3.x

Install dependency:

pip install pygame

## How to Run

python main.py

## Controls

Mouse Left Click → Add obstacle\
R → Reset grid\
O → Generate random obstacles\
1 → Select A\*\
2 → Select Greedy BFS\
M → Manhattan heuristic\
E → Euclidean heuristic\
SPACE → Start search

## Visualization

Red → Start\
Green → Goal / Final Path\
Blue → Visited Nodes\
Yellow → Frontier\
Black → Obstacles

## Metrics

-   Nodes Visited
-   Path Cost
-   Execution Time (ms)
